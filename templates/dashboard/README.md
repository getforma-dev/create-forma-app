# {{PROJECT_NAME}}

A data dashboard built with [Forma](https://getforma.dev).

Features a sortable data table powered by `createList` with fine-grained
reactivity — no virtual DOM diffing required.

## Development

```bash
# Build frontend
cd admin && npm install && npm run build && cd ..

# Run server
cargo run
# Open http://127.0.0.1:3000
```
